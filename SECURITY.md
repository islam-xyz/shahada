# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Shahada.org, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email us at: **islamliveapp@gmail.com**

Include:
- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- A suggested fix (if you have one)

We will acknowledge your report within 48 hours and work to resolve the issue as quickly as possible.

## Scope

This policy applies to:
- The Shahada.org website code in this repository
- Any assets (audio, images, etc.) hosted in this repository

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main branch) | ✅ |
| Older commits | ❌ |

## Our Commitment

- We take security seriously because this site serves people at a vulnerable and important moment in their lives.
- We will never collect personal data, so data breaches are not applicable.
- We will credit security researchers who report valid vulnerabilities (unless you prefer to remain anonymous).

---

Thank you for helping keep Shahada.org safe.

Shahada.org-এ অবদান রাখুন

সবার জন্য ইসলাম গ্রহণের সহজতম গাইডটি আরও ভালো করতে সাহায্য করুন।

সাহায্যের উপায়

  • অনুবাদ সংশোধন করুন বা প্রস্তাব করুন — আমাদের ৪০+ ভাষার একটি বলতে পারলে এবং ত্রুটি দেখলে, রিপোর্ট করুন।
  • বাগ রিপোর্ট করুন — একটি বাগ রিপোর্ট খুলুন।
  • উন্নতির প্রস্তাব দিন — একটি ফিচার অনুরোধ খুলুন।
  • অ্যাক্সেসিবিলিটি উন্নত করুন — অবদান গাইড দেখুন।

কোথা থেকে শুরু করবেন

আমরা সব কাজের জন্য GitHub ব্যবহার করি।

  • ভালো প্রথম ইস্যু
  • সাহায্য চাই
  • সব খোলা ইস্যু
  • খোলা পুল রিকোয়েস্ট

কথা ছড়িয়ে দিন

আপনি যা করতে পারেন তার মধ্যে সবচেয়ে প্রভাবশালী হলো আরও বেশি মানুষকে Shahada.org খুঁজে পেতে সাহায্য করা।

আপনার ওয়েবসাইট থেকে লিংক করুন

আপনি যদি একটি মসজিদের ওয়েবসাইট, ইসলামিক ব্লগ বা কমিউনিটি পেজ চালান, দয়া করে আমাদের লিংক করুন:

<a href="https://shahada.org">Shahada.org — Free guide to becoming Muslim</a>

সোশ্যাল মিডিয়ায় শেয়ার করুন

আপনার সোশ্যাল মিডিয়া অ্যাকাউন্টে Shahada.org শেয়ার করুন:

"Shahada.org ইসলাম গ্রহণের একটি বিনামূল্যে, সহজ গাইড — ৪০+ ভাষায় উপলব্ধ। আগ্রহী যে কারো সাথে শেয়ার করুন। https://shahada.org"

আপনার স্থানীয় মসজিদকে বলুন

আপনার ইমাম বা কমিউনিটি লিডারকে ইসলামে আগ্রহী মানুষদের জন্য একটি রিসোর্স হিসেবে Shahada.org উল্লেখ করতে বলুন।

ইসলামিক রিসোর্স ডিরেক্টরিতে যোগ করুন

আপনি যদি ইসলামিক রিসোর্স তালিকা জানেন, দয়া করে Shahada.org যোগ করার প্রস্তাব দিন।

অবদান রাখার আগে

দয়া করে আমাদের অবদান গাইড এবং আচরণবিধি পড়ুন।

প্রশ্ন?

একটি GitHub issue খুলুন এবং জিজ্ঞাসা করুন।

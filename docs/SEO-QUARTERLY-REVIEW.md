# SEO & backlinks — quarterly review

Run this checklist **every three months** (set a calendar reminder). It complements [SEO-GSC-BASELINE.md](SEO-GSC-BASELINE.md) and [SEO-OUTREACH-PROSPECTS.md](SEO-OUTREACH-PROSPECTS.md).

## 1. Google Search Console (30 minutes)

- [ ] **Performance:** Compare last 3 months to prior period — impressions, clicks, average position for queries containing: `shahada`, `convert`, `islam`, `moslim`, `bekeren`.
- [ ] **Links report:** Note new **external** domains linking to you; investigate any sudden spikes (spam) via disavow only if clearly toxic and after careful review.
- [ ] **Pages / indexing:** Count indexed vs excluded; spot-check any spike in “Crawled — not indexed”.
- [ ] **Manual actions / Security issues:** Must be clean.

## 2. Outreach (ongoing)

- [ ] Send **2–3 small batches** of personalized emails (5–15 contacts per batch) from [SEO-OUTREACH-PROSPECTS.md](SEO-OUTREACH-PROSPECTS.md); refresh the list with new mosques, student groups, or NGOs.
- [ ] Track responses in a simple spreadsheet: organization, date, outcome (linked / declined / no reply).

## 3. On-site sanity check (15 minutes)

- [ ] Homepage still returns **200** for `/en/` and priority locales.
- [ ] `https://shahada.org/sitemap.xml` loads and includes new important URLs.
- [ ] `https://shahada.org/robots.txt` still allows crawling and references the sitemap.

## 4. Competitive reality check

- [ ] Search 3–5 target queries in an incognito window (or Search Console’s query list) and note **who ranks above you** (Wikipedia, governments, large orgs). Adjust expectations and content gaps accordingly — backlinks alone rarely beat entrenched authorities on every head term.

## 5. Red lines

- Do **not** buy bulk links or use automated outreach tools that spam.
- Do **not** guarantee rankings to stakeholders; report **trends** (impressions, clicks, quality links earned).

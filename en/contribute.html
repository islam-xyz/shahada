Contribute to Shahada.org

Help make the world’s simplest guide to embracing Islam better for everyone. We welcome volunteers — developers, translators, and anyone who cares about clarity and accuracy.

Ways to help

  • Fix or suggest translations — If you speak one of our 40+ languages and see an error, report it or submit a pull request.
  • Report bugs — Something broken on your device or browser? Open a bug report.
  • Suggest improvements — Have an idea for content or UX? Open a feature request.
  • Improve accessibility — Help us reach people with disabilities. Check our Contributing guide and open an issue or PR.

Where to start

We use GitHub for all tasks. You don’t need to be a developer to contribute — many tasks are translations or small edits.

  • Good first issues — Small, well-defined tasks for new contributors.
  • Help wanted — Tasks where we’d love community help.
  • All open issues — Bugs, features, and ideas.
  • Open pull requests — See what others are working on.

Before you contribute

Please read our Contributing guide and Code of Conduct. We keep the project simple (plain HTML, CSS, JS) so anyone can contribute.

Questions?

Open a GitHub issue and ask. We’ll get back to you as soon as we can.
